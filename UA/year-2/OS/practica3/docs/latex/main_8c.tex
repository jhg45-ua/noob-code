\doxysubsection{Descripción detallada}
Punto de entrada del simulador de gestión de memoria. 

Implementa dos interfaces de usuario\+:
\begin{DoxyItemize}
\item {\bfseries{GUI}}\+: Interfaz gráfica con Raylib (proceso hijo)
\item {\bfseries{TUI}}\+: Interfaz de terminal para depuración (proceso padre)
\end{DoxyItemize}

Usa fork() para ejecutar ambas interfaces en paralelo.
