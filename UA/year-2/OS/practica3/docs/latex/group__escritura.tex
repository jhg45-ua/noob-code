int write\+\_\+all (\begin{DoxyParamCaption}\item[{int}]{fd}{, }\item[{const void \texorpdfstring{$\ast$}{*}}]{buffer}{, }\item[{size\+\_\+t}]{size}{}\end{DoxyParamCaption})}



Escribe todos los bytes del buffer al descriptor de fichero. 

Garantiza que se escriban exactamente {\ttfamily size} bytes, manejando automáticamente las escrituras parciales que pueden ocurrir con pipes, sockets o cuando el kernel interrumpe la operación.


\begin{DoxyParams}[1]{Parámetros}
\mbox{\texttt{in}}  & {\em fd} & Descriptor de fichero de destino (debe estar abierto para escritura) \\
\hline
\mbox{\texttt{in}}  & {\em buffer} & Puntero a los datos a escribir \\
\hline
\mbox{\texttt{in}}  & {\em size} & Cantidad de bytes a escribir\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Devuelve}
0 si se escribieron todos los bytes correctamente 

-\/1 si hubo error (errno contiene el código de error)
\end{DoxyReturn}
\begin{DoxyPrecond}{Precondición}
fd debe ser un descriptor válido abierto para escritura 

buffer != NULL 

size \texorpdfstring{$>$}{>} 0
\end{DoxyPrecond}
\begin{DoxyNote}{Nota}
Maneja automáticamente EINTR (interrupciones del sistema) 

Reintenta automáticamente en caso de escrituras parciales
\end{DoxyNote}
