\doxysubsection{Descripción detallada}
Librería de utilidades para operaciones de E/S con descriptores de fichero. 

Este módulo proporciona funciones robustas para\+:
\begin{DoxyItemize}
\item Lectura/escritura completa de datos (manejo de escrituras parciales)
\item Transferencia de datos entre descriptores (archivos, pipes, sockets)
\item Copia de archivos
\end{DoxyItemize}

Todas las funciones manejan correctamente\+:
\begin{DoxyItemize}
\item Interrupciones del sistema (EINTR)
\item Escrituras/lecturas parciales
\item Condiciones de EOF
\end{DoxyItemize}
