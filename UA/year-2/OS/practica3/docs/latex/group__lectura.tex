ssize\+\_\+t read\+\_\+all (\begin{DoxyParamCaption}\item[{int}]{fd}{, }\item[{void \texorpdfstring{$\ast$}{*}}]{buffer}{, }\item[{size\+\_\+t}]{size}{}\end{DoxyParamCaption})}



Lee hasta {\ttfamily size} bytes de un descriptor a un buffer. 

Intenta leer exactamente {\ttfamily size} bytes, manejando lecturas parciales y condiciones de EOF. Funciona con cualquier tipo de descriptor\+: archivos regulares, pipes, sockets, etc.


\begin{DoxyParams}[1]{Parámetros}
\mbox{\texttt{in}}  & {\em fd} & Descriptor de fichero de origen (debe estar abierto para lectura) \\
\hline
\mbox{\texttt{out}}  & {\em buffer} & Destino donde se almacenarán los datos leídos \\
\hline
\mbox{\texttt{in}}  & {\em size} & Cantidad máxima de bytes a leer\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Devuelve}
Número de bytes realmente leídos (puede ser menor que size si se alcanzó EOF) 

0 si se alcanzó EOF antes de leer ningún byte 

-\/1 si hubo error (errno contiene el código de error)
\end{DoxyReturn}
\begin{DoxyPrecond}{Precondición}
fd debe ser un descriptor válido abierto para lectura 

buffer != NULL 

size \texorpdfstring{$>$}{>} 0
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondición}
Los datos leídos se almacenan en buffer\mbox{[}0..retorno-\/1\mbox{]}
\end{DoxyPostcond}
\begin{DoxyNote}{Nota}
Maneja automáticamente EINTR (interrupciones del sistema) 

Compatible con buffers, structs o arrays de cualquier tipo
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
El buffer debe tener espacio suficiente para {\ttfamily size} bytes
\end{DoxyWarning}
