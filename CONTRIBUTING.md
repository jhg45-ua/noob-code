# Contribución al proyecto

¡Gracias por considerar contribuir a este proyecto! Valoramos y agradecemos tu tiempo y esfuerzo para mejorar este repositorio. A continuación, encontrarás algunas pautas y recomendaciones para contribuir de manera efectiva.

## Cómo contribuir

1. Haz un fork de este repositorio.
2. Clona tu fork en tu máquina local.
3. Crea una nueva rama para tus cambios: `git checkout -b nombre-de-la-rama`.
4. Realiza los cambios y realiza commits descriptivos de tus adiciones o modificaciones: `git commit -m "Descripción de los cambios"`.
5. Sube tus cambios a tu fork: `git push origin nombre-de-la-rama`.
6. Abre una solicitud de extracción (pull request) explicando tus cambios de manera clara y concisa.

## Pautas para contribuir

- ~~Asegúrate de seguir los estándares de codificación y las convenciones de estilo del proyecto.~~
- Documenta cualquier cambio en el código que realices.
- Mantén las discusiones y comentarios constructivos.

## Reporte de problemas

Si encuentras algún problema o tienes alguna sugerencia, por favor abre un problema en la sección de issues y proporciona la mayor cantidad de información posible para ayudarnos a entender y resolver el problema.

Gracias de nuevo por tu interés en contribuir a este proyecto. ¡Esperamos ver tus aportes pronto!
